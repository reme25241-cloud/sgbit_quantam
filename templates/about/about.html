About the Project

Sensor Module Classification System

This web application classifies various sensor modules from uploaded images using a trained Convolutional Neural Network (CNN) model. Once a sensor is identified, it displays relevant technical details like usage, advantages, disadvantages, wiring, sample code, and more.

ðŸ”§ System Workflow:
  • Step 1: User uploads an image of a hardware sensor module (e.g., PM2.5, PIR, RFID).
  • Step 2: The image is preprocessed and passed to a CNN model trained to classify sensor types.
  • Step 3: Once identified, related information about the sensor (e.g., use, wiring, code) is retrieved and displayed.
  • Step 4: Transformed image outputs (e.g., grayscale, thresholded, equalized) and prediction results are saved to the database.
ðŸ§± Project Structure:
  • Backend: Django with SQLite for storage
  • Model: Custom-trained CNN using labeled sensor images
  • Image Processing: Applies grayscale, Gaussian blur, histogram equalization, thresholding, ROI extraction
  • Database: Saves predictions, transformations, and sensor metadata
  • User Interface: Displays original + transformed images with prediction result
